Git Test
  • git clone - path repository
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить новый файл под контроль Git
  • git commit -am "add new file index.html" - зафиксировать изменения и дать им комментарий
  • git push - отправить все законченные изменения на сервер
  • git pull - забрать свежие обновления
